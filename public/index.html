This is an example Node + Express + Mongoose project starter.

It assumes you have a local database named sample_database served at 127.0.0.1:27017.

You can change the configuration in the .env file.

If you don't have a database locally, you can create it using the following MDB CLI command:

$ mdb database init -db mongodb

... and then provide the configuration that has been displayed in your terminal.
